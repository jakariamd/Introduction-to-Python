# How to install Python 

পাইথন একটি ওপেন সোর্স ল্যাঙ্গুয়েজ প্যাকেজ, যা বিনামূল্যে পাওয়া যায়। এটি একটি General Purpose Language, যা যেকোনো ছোট কিংবা বড ডেভেলপমেন্ট এ ব্যবহার করা যায়। পাইথন সম্পর্কে বিস্তারিত পাওয়া যাবে নিচের লিংকগুলোতে। 

  - **Download:** https://www.python.org/
  - **Documentation:** https://docs.python.org/3/ 


পাইথনের ওয়েবপেজ এ এর অনেকগুলি ভার্সন পাওয়া যায়। রিকমেন্ডেড হলো Latest ভার্সন ডাউনলোড না করে এর কয়েক ভার্সন আগেরটা ডউনলোড করা। যেমন পাইথনের বর্তমান Latest ভার্সন হচ্ছে [Python 3.8.2](https://www.python.org/ftp/python/3.8.2/Python-3.8.2.tgz)। আমি রিকমেন্ড করবো [Python 3.8.5](https://www.python.org/ftp/python/3.6.10/Python-3.6.10.tgz)।
